# Banua Cloud Website

Website ini adalah halaman resmi Banua Cloud untuk menampilkan profil perusahaan dan informasi layanan digital.

Saat ini proyek digunakan sebagai **halaman “Maintenance / Coming Soon”** agar pengunjung tetap mendapat informasi yang jelas selama website utama masih disiapkan.

## Tujuan Proyek

- Menjadi identitas online resmi Banua Cloud.
- Memberikan informasi singkat tentang layanan utama perusahaan.
- Menyediakan jalur kontak cepat (email, telepon, dan WhatsApp).
- Menjaga komunikasi dengan pengunjung selama masa pengembangan website penuh.

## Gambaran Isi Halaman

- Branding Banua Cloud (nama, tagline, dan logo).
- Pesan bahwa website sedang dalam proses pengembangan.
- Ringkasan layanan inti.
- Informasi kontak dan tautan media sosial (jika diaktifkan).

## Status

Proyek ini masih dalam tahap pengembangan. Setelah website utama siap, halaman maintenance bisa dinonaktifkan dan diganti ke halaman website penuh.
